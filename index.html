Reach is an application that lets you send WhatsApp notifications to your customers.

How to use the app

The template

The app lists the WhatsApp templates that have been approved. The app currently does not support templates with image or document headers, only text headers. First, select the template you would like to use to notify your customers.

The language

A template may be published in different languages. If the template you selected is available is more than one language, you can use the language dropdown to select the language of the template. The content will update accordingly.

The CSV file

A template may contain just a body component or a header, a body and a footer component. Each of those components could contain a parameter that needs to be specified when sending a notification. The next section explains how to build the CSV file depending on the template.

The number of notifications is limited to 500 per file.

When there are no parameters, the CSV must contain only one column and each row must have only one phone number.

Example:

                    
      7188765678
      1230987687

When there are parameters in the template, the CSV must contain one column per parameter.

  • The first column must be the phone number.
  • The next columns must be the variables of the header.
  • The next columns must be the variables of the body.

Example with the following template:

HEADER: Hi {{1}}.

BODY: This is a message from {{1}} on {{2}}.

FOOTER: Thank you.

                    
      7188765678, Rose, Your Support Advisor, 01-01-2022
      1230987687, Jack, Your Support Advisor, 01-01-2022
