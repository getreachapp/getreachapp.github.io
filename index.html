How to setup the app

In order to setup the Reach App, you need 3 things: App ID, App Token and the WhatsApp namespace.

App ID

In your Sunshine Conversation dashboard, click on your app. Navigate to settings and copy your App ID.

Send bulk templates

Send bulk proactive WhatsApp messages under "Send bulk notifications via CSV".

First, select a template and the language from the dropdowns. Then, build a CSV file for the selected template. Below are the instructions for building the CSV file depending on the template variables.

Template without variables

When there are no parameters, the CSV must contain only one column and each row must have only one phone number.

Example:

                    
      7188765678
      1230987687

Template with variables

When there are parameters in the template, the CSV must contain one column per parameter.

  • The first column must be the phone number.
  • The next columns must be the variables of the header.
  • The next columns must be the variables of the body.

Example with the following template:

HEADER: Hi {{1}}.

BODY: This is a message from {{1}} on {{2}}.

FOOTER: Thank you.

                    
      7188765678, Rose, Your Support Advisor, 01-01-2022
      1230987687, Jack, Your Support Advisor, 01-01-2022
